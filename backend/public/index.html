Auth (01)
POST
/api/auth/login Authenticate a user with email & password. Returns a JWT access token and user profile.
Request Body
{
  "email":    "user@example.com",
  "password": "password123"
}
Response
{
  "message": "Logged in successfully",
  "token":   "JWT Token"
}
POST
/api/auth/register Create a new user account. Returns the created user object and an authentication token.
Request Body
{
  "username": "user",
  "email":    "user@example.com",
  "password": "password123"
}
Response
{
  "message": "User registered successfully",
  "token":   "JWT Token"
}
Accounts (02)
GET
/api/accounts/ Retrieve a list of all bank accounts linked to the authenticated user.
Headers
Authorization: Bearer <token>
GET
/api/accounts/:id Retrieve a single bank account by its unique identifier.
Headers
Authorization: Bearer <token>
DELETE
/api/accounts/:id Permanently remove a bank account by its unique identifier. Action is irreversible.
Headers
Authorization: Bearer <token>
Transactions (03)
GET
/api/transactions Fetch all transactions across all linked bank accounts for the authenticated user.
Headers
Authorization: Bearer <token>
GET
/api/transactions/:accountId Fetch all transactions belonging to a specific bank account by its account ID.
Headers
Authorization: Bearer <token>
GET
/api/transactions/recent Retrieve the last 10 transactions across all accounts. Used on the homepage dashboard.
Headers
Authorization: Bearer <token>
Transfers (04)
POST
/api/transfer/send Initiate a payment transfer between two bank accounts. Requires sender ID, receiver ID, and amount.
Headers
Authorization: Bearer <token>
Homepage (05)
GET
/api/data Aggregate dashboard data — returns user profile, all accounts, transactions, total balance, and recent transactions in a single call.
Headers
Authorization: Bearer <token>
Response Shape
{
  "user":             User,
  "accounts":         Account[],
  "recentTransactions": Transaction[]
}
Plaid (06)
POST
/api/create-link-token Generate a Plaid Link token to initialize the Plaid Link flow for connecting a bank account.
Headers
Authorization: Bearer <token>
Request Body
{
  "userId": "<userId>"
}
Response
{
  "link_token": "link-sandbox-..."
}
POST
/api/exchange-public-token Exchange a short-lived public token (from Plaid Link) for a permanent access token to access account data.
Headers
Authorization: Bearer <token>
Request Body
{
  "publicToken": "<publicToken>"
}
Response
{
  "message":      "Plaid account linked successfully",
  "access_token": "access-sandbox-..."
}
